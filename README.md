# Султан 90-х | Карточная игра

Браузерная многопользовательская карточная игра с рейтинговой системой.

## Установка и запуск

npm install
npm start

Откройте в браузере: http://localhost:3000

## Как играть

1. Регистрация или вход
2. Выберите игрока из списка онлайн
3. Нажмите "Пригласить вдвоём" или "Пригласить втроём"

## Правила

- Пики бьются только пиками старше
- Буби (козырь) бьют любую карту другой масти
- Дама заканчивает кон

## Технологии

- Backend: Node.js, Express, Socket.io
- Frontend: HTML, CSS, JavaScript
